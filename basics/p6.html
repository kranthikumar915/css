defining color value

color in css most often specifies in the foloowing formats:

  1. a color keyword - like - "red" , "green" , "blue" ,transparent" etc
  2. a hex value - like "#ff0000" , "#00ff00", etc
  3. a rgb values (red, green,blue) - like "rgb(225,0,0)"
