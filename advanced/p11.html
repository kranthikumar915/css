CSS PSEUDO ELEMENTS...!

the css pseudo elements allows you to the style elements or parts of the elements without adding any id's or classes to them

CSS3 introduced a new double-colon(::) syntax for pseudo elements to distiguish between them and pseudo classes

selector::pseudo element {property : value;}
