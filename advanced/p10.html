CSS-PSEUDO CLASS

the css pseudo class allows you to style the dynamic states of an element such as hover active and focus state as well as elements that are existing in the document tree but cant be targeted via the use of other selectors without adding any id or class to them for example targeting the first or last child elements

a pseudo class starts with a colon(:) its syntax can be given with:

selector:pseudo class {property : value;}
